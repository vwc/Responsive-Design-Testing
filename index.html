Darstellung von Webseiten auf mobilen Endgeräten

Diese Seite vermittelt Ihnen einen Eindruck, wie unterschiedlich eine Webseite auf verschiedenen mobilen Endgeräten dargestellt wird.

Da mittlerweile auf der Couch und vor dem Fernseher oder im Bett via Smartphone im Internet gesurfed wird, muss bei der Erstellung einer Webseite von Beginn an die unterschiedliche Darstellung bedacht werden. Geben Sie die gewünschte URL ein, um das Erscheinungsbild zu testen

240 x 320 (small phone)

320 x 480 (iPhone)

480 x 640 (small tablet)

768 x 1024 (iPad - Portrait)

1024 x 768 (iPad - Landscape)
